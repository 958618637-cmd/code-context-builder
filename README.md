## What is Continue?

> _Note: The `continuedev/continue` repository is no longer actively maintained and is read-only for all users._

Continue is a coding agent available as a [VS Code extension](#vs-code) and [JetBrains plugin](#jetbrains).

## Final 2.0.0 Release

We polished Continue and did a final 2.0.0 release of the VS Code extension and JetBrains plugin.

This included removing anonymous telemetry, pulling out authentication, squashing bugs, and more.
